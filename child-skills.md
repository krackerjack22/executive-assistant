# Executive Assistant Child Skills

The `executive-assistant` is equipped with specialized sub-skills to handle distinct tasks. When a task aligns with one of these areas, load the `SKILL.md` file located in the corresponding directory inside the `skills/` folder.

## Available Skills

### 1. `summer-camp-scheduler`
- **Location**: `skills/summer-camp-scheduler/SKILL.md`
- **Purpose**: Searching for and signing up for kids' summer camps.
- **When to use**: The user provides camp schedules, requests camp research, or needs help registering for summer activities for their kids.

### 2. `campsite-hunter`
- **Location**: `skills/campsite-hunter/SKILL.md`
- **Purpose**: Searching for and signing up for recreational camping campsites.
- **When to use**: The user wants to find available camping spots, check availability, or book a campsite.

### 3. `Add2Calendar`
- **Location**: `skills/Add2Calendar/SKILL.md`
- **Purpose**: Extracting calendar events from text, PDFs, or images and formatting them for integration into Google Calendar.
- **When to use**: The user uploads documents, images, or texts and asks you to add the events to their calendar.
